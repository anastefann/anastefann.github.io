THE LONG DARK REVERSE ENGINEERED UI SYSTEM

UI & Technical Game Designer

GENERAL GAME INFORMATION

This project contains menus and a few modular assets (widgets) that can be used to create main menus. For this project, I replicated The Long Dark's UI as part of a reverse engineering project, but if the assets are replaced, it can easily be used for personal projects as well.

  • Genre: Modular UI Menu Pack
  • Engine: Unreal Engine 5
  • Team Size: Solo Project
  • Duration: 7 weeks
  • Platform: PC
MY RESPONSIBILITIES

Implemented The Long Dark's UI Menus Into Unreal Engine 5

I made use of Unreal Motion Graphics (UMG) for the UI implementation, having horizontal and vertical boxes to organize the menus seamlessly. Incorporating animations, I showcased the axe on the left side of a button and achieved a smooth fading-in effect upon hovering over it. The Unreal hover events proved particularly valuable in efficiently implementing different descriptions for the buttons. Furthermore, I incorporated sliders for options such as audio and buttons to go between multiple options throughout the settings menus to streamline the process of making selections.

Achievements

In navigating the challenges of UI implementation as a beginner, I embraced the learning curve, allowing room for mistakes that ultimately enhanced my understanding. Despite initially planning to recreate various in-game elements, time constraints led to a strategic scope down, focusing solely on the main menus within a limited three-week timeframe. The use of modular assets proved instrumental in efficiently constructing menus, particularly inspired by The Long Dark's consistent design. Early research documentation and a How To Document ensured a comprehensive and organized approach to UI creation, fostering a thoughtful consideration of in-engine elements.
